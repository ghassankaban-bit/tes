Materi Dasar Python

Pelajari konsep fundamental dengan contoh kode yang bisa Anda coba.

1. Variabel & Tipe Data

Variabel adalah wadah untuk menyimpan nilai data. Python memiliki berbagai tipe data seperti string (teks), integer (bilangan bulat), dan float (bilangan desimal).

# Variabel untuk menyimpan nama
nama = "Budi"
# Variabel untuk menyimpan umur
umur = 21
# Variabel untuk menyimpan tinggi badan
tinggi = 175.5

print(f"Nama: {nama}, Tipe: {type(nama)}")
print(f"Umur: {umur}, Tipe: {type(umur)}")
print(f"Tinggi: {tinggi}, Tipe: {type(tinggi)}")

2. Struktur Kontrol: If-Else

Pernyataan `if-else` digunakan untuk membuat keputusan dalam kode. Program akan menjalankan blok kode tertentu jika suatu kondisi terpenuhi (benar) dan blok lain jika tidak.

nilai = 85

if nilai >= 90:
    print("Predikat: A")
elif nilai >= 80:
    print("Predikat: B")
elif nilai >= 70:
    print("Predikat: C")
else:
    print("Predikat: D")

3. Perulangan (Loops)

Perulangan `for` digunakan untuk mengulang eksekusi blok kode beberapa kali. Ini sangat berguna untuk iterasi melalui item dalam sebuah list, string, atau objek iterable lainnya.

# Mengulang angka dari 0 hingga 4
print("Mengulang angka:")
for i in range(5):
    print(f"Angka: {i}")

# Mengulang item dalam sebuah list
print("\nMengulang buah-buahan:")
buah = ["apel", "jeruk", "mangga"]
for item in buah:
    print(f"Saya suka {item}")
